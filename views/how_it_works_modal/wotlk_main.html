No more downranking

The need for downranking was removed in Patch 3.0.2, the release patch for Wrath of the Lich King. The patch introduced a change that causes all spells to cost a percentage of the casters base mana, independent of spell rank, thereby completely voiding the purpose of downranking.

This website has thus run its course, I will keep it up for as long as I can for those playing older expansions or for general interest.

Thanks for the run and see you in another 15 years!
